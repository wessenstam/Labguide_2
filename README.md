# Labguide 2 — Computer Conversations

Hands-on **Computer Conversations** lab material for DevRev (part **2** of the trilogy): using **agentic AI** on the **CX** side to answer customer questions via agents, the Computer for End-users plugin, a Chrome Extension, and workflows.

## What's in this repo

- **`docs/`** — Markdown lab pages (screenshots under `docs/images/`).
- **`mkdocs.yml`** — [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) site config.

Modules cover **creating and configuring an agent** (prompt, skills, knowledge), **Computer for End-users**, a **Chrome Extension** demo, **workflow export/import/configuration**, **customer conversations**, and **workflow log troubleshooting**.

This workshop can be run standalone but depends on knowledge-base articles from [Labguide 1](https://github.com/wessenstam/Labguide_1).

## Local preview

```bash
pip install mkdocs-material
mkdocs serve
```

Open the URL shown in the terminal (e.g. `http://127.0.0.1:8000`).
