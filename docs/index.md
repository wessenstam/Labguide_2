## Foreword

Welcome to the **second** part of the Computer Conversations: a hands-on exploration of one of the most transformative shifts in how we work. 

We're thrilled you're here. This lab guide represents months of collaboration from our EMEA Solutions Engineering team, and it's been designed with you in mind—to give you practical, real-world experience with AI agents that are reshaping enterprise technology. Thank you for making the time to be part of this journey. What you'll learn over the coming hours isn't just about technology; it's about reimagining what's possible when human intelligence and artificial intelligence work in genuine partnership.

In this second part of the wider workshop we will be focusing on the CX side of DevRev. How can we use Agentic AI to answer questions from our Customers.
This workshop CAN be run on its own, but has some dependencies on the first part for the articles that need to be available.

In this lab we will perform the following actions:


1. Create an agent where we will provide:
   
   1. A prompt
   2. Skills
   3. Knowledge
   4. Small update
   5. Preview the agent
   
2. Configure and create the *Computer for End-users* plugin
3. Configure and use a specilized Chrome Extension to mimic what could be
4. Configure the workflow:

    1. Export an existing workflow
    2. Import a workflow
    3. Configure the workflow

5. Have a conversation with the newly created agent via the Chrome Extension
6. Check the log of the workflow for simple troubleshooting


We wish you lots of fun!!

*Martijn Bosschaart*
